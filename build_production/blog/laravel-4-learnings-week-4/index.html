Laravel: Code Bright

There's a new ebook called Code Bright coming up for Laravel 4. Dayle Rees is rewriting his Code Happy book for the newest version of Laravel. Expect noting less than awesome tutorials on learning how to use Laravel 4.

Dayle also updated his website recently where he blogs frequently about Laravel.
